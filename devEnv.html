BIT@Home 项目开发环境搭建

  • Ubuntu / ROS 开发环境安装
    略。

  • 工作区搭建

  1. mkdir bitathome && cd bitathome
  2. mkdir src
  3. catkin_init_workspace
  4. echo "source 工作区路径/devel/setup.zsh" >> ~/.zshrc
  • 仓库拉取项目源码
  1. cd bitathome/src
  2. git clone https://github.com/ihainan/bitathomeSrc.git

  • 运动控制

    1. 将旅行者三号的数据传输线插入到电脑的 USB 接口上，此时 /dev 目录下会生成 ttyUSB0 文件；
    2. 执行命令： usermod -a -G dialout 你的用户名；
    3. 重启系统；
    4. 确保数据传输线已经插到了电脑的 USB 接口上，运行 roscore；
    5. 打开旅行者三号，执行命令：rosrun bitathome_hardware_control hc_cmd_interface.py;
    6. 如果想要通过键盘控制机器运动，运行 rosrun bitathome_hardware_control hc_vertor_keyboard.py，通过 wasdzxc 按键控制运动;

  • Kinect 视觉 / 骨架检测

    1. 安装 glut-dev ： sudo apt-get install freeglut3-dev；
    2. 从团队 FTP 上下载 /tools/DevLibs/nite-bin-linux-x64-v1.5.2.21.tar.bz2，解压缩，进入目录，执行：sudo ./install.sh;
    3. 进入 bitathome/src 目录，执行 git clone https://github.com/ros-drivers/openni_tracker.git;
    4. 进入 openni_tracker 目录，执行 git checkout hydro-devel；
    5. 回到 bitathome/src 目录，执行 git clone https://github.com/pirobot/skeleton_markers.git
    6. 进入 skeleton_markers 目录，执行 git checkout hydro-devel；
    7. 回到 bitathome 目录，执行 catkin_make。

  • 更新仓库

    1. 在执行修改之前，先更新本地仓库 : git pull;
    2. 完成修改后，添加修改：git add 你所写的package路径；
    3. 提交修改：git commit -m "你修改内容的说明"；
    4. 上传修改：git push origin master。
